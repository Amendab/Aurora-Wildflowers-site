Aurora Wildflowers by Amenda - Downloadable Static Website
Business started in 2026

How to use:
1. Unzip the folder.
2. Open index.html in a browser to preview the website.
3. Upload the whole folder to any static website host, or send it to a web designer.

Files:
- index.html: website content and page sections
- styles.css: colours, layout, fonts and mobile design
- script.js: catalog products, prices, filters and quote message builder
- assets/images: optimized product photos and logo

How to edit prices/catalog:
Open script.js and edit the products array and priceRows array near the top.

Suggested note:
Listed prices are simple CAD starting prices and are kept under $200. Update them anytime as needed.
